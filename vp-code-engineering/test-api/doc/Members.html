Members

Create a new member by providing his/her name, address and telephone number. You will receive an object that holds the name, address, telephone and member ID of the member.

Resource URL
http://localhost:5000/members
Curl Example
$ curl -X POST -d '{"name":"Peter","address":"100, Happy Valley","telephone":"12345678"}' -H 'content-type: application/json' -H 'authorization: Bearer mytoken123' "http://localhost:5000/members"
Request
Member
NameTypeDescription
name String The full name of the member.
address String

The residential address of the member.

telephone String

The telephone number of the member.

Request Header
content-type: application/json authorization: Bearer mytoken123
Request Body
{ "name": "Peter", "address": "100, Happy Valley", "telephone": "12345678" }
Response - Status 200
MemberID
NameTypeDescription
value String

The member ID.

Response Header
content-type: application/json; charset=utf-8 status: 200 OK ratelimit-limit: 1200 ratelimit-remaining: 1137 ratelimit-reset: 1415984218
Response Body
{ "MemberID": { "value": "MB-3001" } }
